design twitter

Problem Description

Design a simplified version of Twitter where users can post tweets, follow/unfollow another user and is able to see the 10 most recent tweets in the user’s news feed. Your design should support the following methods:
1.postTweet(userId, tweetId): Compose a new tweet.
2.getNewsFeed(userId): Retrieve the 10 most recent tweet ids in the user’s news feed. Each item in the news feed must be posted by users who the user followed or by the user herself. Tweets must be ordered from most recent to least recent.
3.follow(followerId, followeeId): Follower follows a followee.
4.unfollow(followerId, followeeId): Follower unfollows a followee.

Example:

Twitter twitter = new Twitter();

// User 1 posts a new tweet (id = 5).
twitter.postTweet(1, 5);

// User 1’s news feed should return a list with 1 tweet id -> [5].
twitter.getNewsFeed(1);

// User 1 follows user 2.
twitter.follow(1, 2);

// User 2 posts a new tweet (id = 6).
twitter.postTweet(2, 6);

// User 1’s news feed should return a list with 2 tweet ids -> [6, 5].
// Tweet id 6 should precede tweet id 5 because it is posted after tweet id 5.
twitter.getNewsFeed(1);

// User 1 unfollows user 2.
twitter.unfollow(1, 2);

// User 1’s news feed should return a list with 1 tweet id -> [5],
// since user 1 is no longer following user 2.
twitter.getNewsFeed(1);

My Code



class Twitter(object):


    def initUser(self,userId):
        self.userIdList.append(userId)
        self.followIdDict[userId] = [userId]



    def __init__(self):
        """
        Initialize your data structure here.
        """
        self.userIdList = []    #all userId
        self.followIdDict = {}  #userId:all followee id
        self.tweetsQueue = []   #[userId,tweets id]



    def postTweet(self, userId, tweetId):
        """
        Compose a new tweet.
        :type userId: int
        :type tweetId: int
        :rtype: void
        """
        if userId not in self.userIdList:
            self.initUser(userId)

        self.tweetsQueue.append([userId,tweetId])




    def getNewsFeed(self, userId):
        """
        Retrieve the 10 most recent tweet ids in the user's news feed. 
        Each item in the news feed must be posted by users who the user followed or by the user herself. 
        Tweets must be ordered from most recent to least recent.
        :type userId: int
        :rtype: List[int]
        """
        if userId not in self.userIdList:
            return []

        tempTweets = []
        len_t_q = len(self.tweetsQueue)+1
        count = 0

        for i in range(1,len_t_q):
            if count >= 10:
                break
            tweet = self.tweetsQueue[-1*i]

            if tweet[0] in self.followIdDict[userId]:
                tempTweets.append(tweet[1])
                count += 1

        return tempTweets 





    def follow(self, followerId, followeeId):
        """
        Follower follows a followee. If the operation is invalid, it should be a no-op.
        :type followerId: int
        :type followeeId: int
        :rtype: void
        """
        if followerId not in self.userIdList:
            self.initUser(followerId)
        if followeeId not in self.userIdList:
            self.initUser(followeeId)

        if followeeId not in self.followIdDict[followerId]:    
            self.followIdDict[followerId].append(followeeId)



    def unfollow(self, followerId, followeeId):
        """
        Follower unfollows a followee. If the operation is invalid, it should be a no-op.
        :type followerId: int
        :type followeeId: int
        :rtype: void
        """
        if (followerId in self.userIdList) and (followeeId in self.userIdList) and 
               (followeeId in self.followIdDict[followerId]) and (followeeId != followerId):
            self.followIdDict[followerId].remove(followeeId)

Code is simple, but I it took me a lot of time to fix bugs.

First, I did not design the system well at the beginning. I designed a dictionary variable tweetDict, in which the element is userId(key):tweetList(value), to keep tweets for each user. But then I found that it is difficult to remove or add tweets when the user follow or unfollow someone. Then I thought about switching this variable to a new form: to let the value be a list [followeeId, tweetList] so that I can record the owner of the tweet. And I should keep track of both one’s followers and followees. At this time, I felt I have fallen into the abyss. Each time the user follows or unfollows someone, all variables should change. What’s worse, the change means traversing the whole list. So I think of another way to solve it: record all tweets in one list and only when it calls getNewsFeed() the list needs to be visited. Note that list is a natural way to record which tweet is an earlier one.

Still, I met with some other problems, due to my poor knowleadge for python class. I define and initialize all class variables outside the constructor, without recognition of them being static variables for python. That’s also why I get different answers when I “run code” and “submit” in leetcode.

There are also some other trival details. For example, the users follow themselves and could not unfollow themselves:)

Like what Song said, I should design before write it!

Intersection of Two Arrays

Problem Description

Given two arrays, write a function to compute their intersection.

Example:
Given nums1 = [1, 2, 2, 1], nums2 = [2, 2], return [2].

Note:
Each element in the result must be unique.
The result can be in any order.

My Code


class Solution(object):
    def intersection(self, nums1, nums2):

        rNums = []        
        nums1 = list(set(nums1))
        nums2 = list(set(nums2))
        nums1.sort()
        nums2.sort()

        ind = 0
        for num1 in nums1:

            for i in range(ind,len(nums2)):
                if num1 == nums2[i]:
                    ind = i + 1
                    rNums.append(num1)
                    break
            continue

        return rNums

Problem Description

Given two arrays, write a function to compute their intersection.

Example:
Given nums1 = [1, 2, 2, 1], nums2 = [2, 2], return [2, 2].

Note:
Each element in the result should appear as many times as it shows in both arrays.
The result can be in any order.

My Code


class Solution(object):
    def intersect(self, nums1, nums2):

        rNums = []
        nums1.sort()
        nums2.sort()

        num1_index = 0
        num2_index = 0

        for i in range(num1_index,len(nums1)):
            for j in range(num2_index,len(nums2)):
                if nums1[i] == nums2[j]:
                    num1_index = i + 1
                    num2_index = j + 1
                    rNums.append(nums1[i])
                    break

        return rNums
